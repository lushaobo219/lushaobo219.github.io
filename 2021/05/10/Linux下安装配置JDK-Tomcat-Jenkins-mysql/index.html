Linux下安装配置JDK+Tomcat+Jenkins+mysql

发表于 2021-05-10 更新于 2021-05-22 阅读次数：
本文字数： 2.2k 阅读时长 ≈ 2 分钟

1，JDK安装

1.1) 新建jdk文件夹（路径：/usr/local/jdk）

1.2) 将jdk安装包解压到jdk文件夹中

tar –xvf jdk-7u71-linux-i586.tar.gz –C /usr/local/jdk/

(下载安装包的时候，注意系统位数，我的是64位，附上下载地址：http://www.oracle.com/technetwork/java/javase/downloads/jdk7-downloads-1880260.html)

1.3) 配置jdk环境变量，打开/etc/profile配置文件，将下面配置拷贝进去

1.4) 重新加载/etc/profile配置文件 source /etc/profile

​ 验证：java –version

​ 有版本号出现，说明jdk安装成功

2，安装Tomcat

2.1) 新建tomcat文件夹（路径：/usr/local/tomcat）

2.1) 将安装包解压到tomcat文件夹中

tar -xvf apache-tomcat-7.0.63.tar.gz -C /usr/local/tomcat/

(附上下载地址:http://tomcat.apache.org/download-70.cgi)

2.3）打开防火墙

2.4）测试tomcat

在本地浏览器地址栏输入：http:localhost:8080

tomcat默认端口为8080，如和其他应用端口号冲突，可在/usr/tomcat/apache-tomcat-7.0.27/conf/server.xml中修改

出现tomcat的页面表示安装成功
