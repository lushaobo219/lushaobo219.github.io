一、Selenium Grid介绍

​ Selenium Grid是一个分布式WebUI测试工具，可以将测试流程分发到多台服务器上，并行地执行。Selenium Grid架构中包含两个主要角色：Hub是中心点控制节点，而Node是Selenium的工作节点，它们注册到Hub上，并会操作浏览器执行由Hub下发的自动测试用例。
